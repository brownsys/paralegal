Crate paralegal_flow

Source
Expand description

Ties together the crate and defines command line options.

While this is technically a “library”, it only is so for the purposes of being able to reference the same code in the two executables paralegal_flow and cargo-paralegal-flow (a structure suggested by rustc_plugin).
