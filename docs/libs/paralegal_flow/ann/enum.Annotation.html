Enum Annotation

Source 
pub enum Annotation {
    Marker(MarkerAnnotation),
    OType(TypeId),
    Exception(ExceptionAnnotation),
}
Expand description

Types of annotations we support.

Usually you’d expect one of those annotation types in any given situation. For convenience the match methods Self::as_marker, Self::as_otype and Self::as_exception are provided. These are particularly useful in conjunction with e.g. Iterator::filter_map
