Expand description

Formating of the tree structure.

Example:

DisplayFmtPadding {
    join_middle: "|-",
    join_last: "'-",
    join_haschild: "-+-",
    join_default: "---",
    indent_middle: "| ",
    indent_last: "  ",
}

will show the tree as

-+- root
 |--+- a
 |  '--+- b
 |     '---- c
 |--+- d
 |  '---- e
 '---- f
