Enum rustc_ast::ast::NestedMetaItem

source · 
pub enum NestedMetaItem {
    MetaItem(MetaItem),
    Lit(MetaItemLit),
}
Expand description

Values inside meta item lists.

E.g., each of Clone, Copy in #[derive(Clone, Copy)].

Variants§

§

MetaItem(MetaItem)

A full MetaItem, for recursive meta items.

§

Lit(MetaItemLit)

A literal.

E.g., "foo", 64, true.
