Module which contains the snapshot/rollback functionality of the ena data structures.

For most usecases this is just an internal implementation detail. However if many ena data structures are snapshotted simultaneously it is possible to use UnificationTableStorage/SnapshotVecStorage instead and use a custom UndoLogs<T> type capable of recording the actions of all used data structures.

Since the *Storage variants do not have an undo log with_log must be called with the unified log before any mutating actions.
